## Homework - Framework Wandering - UIKit Dynamics

Take a look at [this sample project](https://developer.apple.com/library/ios/samplecode/DynamicsCatalog/Introduction/Intro.html) from Apple that shows some of the functionality possible with the UIKit Dynamics classes. Also take a look at [this Ray Wenderlich tutorial](http://www.raywenderlich.com/94719/uikit-dynamics-swift-tutorial-tossing-views) on how to take an image and allow it to be tossed offscreen by the user.

Explore what is possible with this super cool framework and either make a new project that incorporates some of these features or enhance a previous assignment with some behaviors from UIKit Dynamics.

See these class reference docs for additional info:

* [UIDynamicAnimator](https://developer.apple.com/library/ios/documentation/UIKit/Reference/UIDynamicAnimator_Class/index.html#//apple_ref/doc/uid/TP40013153)
	* [UIDynamicAnimator Delegate](https://developer.apple.com/library/ios/documentation/UIKit/Reference/UIDynamicAnimatorDelegate_Protocol/index.html#//apple_ref/doc/uid/TP40013352)
* [UIGravityBehavior](https://developer.apple.com/library/ios/documentation/UIKit/Reference/UIGravityBehavior_Class/index.html#//apple_ref/doc/uid/TP40013155)
* [UICollisionBehavior](https://developer.apple.com/library/ios/documentation/UIKit/Reference/UICollisionBehavior_Class/index.html#//apple_ref/doc/uid/TP40013152)
	* [UICollisionBehavior Delegate](https://developer.apple.com/library/ios/documentation/UIKit/Reference/UICollisionBehaviorDelegate_Protocol/index.html#//apple_ref/doc/uid/TP40013151)
